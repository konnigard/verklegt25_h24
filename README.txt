How To Run:
In your terminal you want to input the file path of the folder that the program is stored in.
You can do this by right clicking the folder and copying the path.
then you want to write "CD" and paste and press enter.
Finally you want to write "python __main__.py" and press enter.

CONGRATULATIONS!!! You've ran our program!
